# Task3ProfileApp

Приложение-профиль на Qt/QML для Android.

Функции:

- круглая фотография пользователя;
- username социальной сети;
- список интересов;
- кнопка «Подписаться» с открытием внешней ссылки;
- уникальные PNG-иконки в папке `icons/`.

Перед сдачей замените `@my_username`, ссылку и `resources/profile.png` на свои данные.
