# Task4GuessNumber

Игра «Угадай число» на Qt/QML для Android.

Реализовано:

- архитектура MVVM: `GameViewModel` + QML View;
- C++-логика игры;
- обработка некорректного ввода;
- хранение настроек через `QSettings`;
- счетчик запусков;
- уникальный ID приложения;
- имя пользователя, число попыток, лучший результат;
- диалог результата;
- анимация кнопки;
- локализация через `qsTr()` и `.ts` файлы;
- unit-тесты в папке `tests/`.
